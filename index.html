HEC University Database System (HUDS)

Serving Information Simply

1. Introduction

Data management helps minimize potential errors by establishing processes and policies for usage and building trust in the data being used
to make decisions across your organization. With reliable, up-to-date data, companies can respond more efficiently to market changes and customer needs.
As there are many Universities that are recognized by the HEC, managing them isn’t a child play. So, we are going to make a database system, named HUDS.

2. Objectives

To Design a well-managed Web-Based application to manage all Universities data that are
recognized by the HEC.

3. Methodology

It will manage all the Teacher, Student, Department and Faculties of the Universities that are recognized by the HEC.
Everything will be managed through that web-based application, like data entry, data retrieving,etc,
There will be a reporting section, that will generate a required report.

4. Project Scope

This project will be applicable in any field that has to manage the data, from Universities to Schools, from Offices to small shops.

5. Solution Application Areas

This project will help everyone that works with the management of data. Database management isn’t a child’s play.
But the user interface we will be giving will be much easy and user friendly. So any employ with basic knowledge of computer will be able to use and manage it.
